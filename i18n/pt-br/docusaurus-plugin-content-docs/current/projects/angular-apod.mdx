## 📋 Pré-requisitos

- Node.js (versão 18 ou superior)
- npm

---

## 🔧 Instalação

1. Clone o repositório:

```bash
git clone https://github.com/nixoletas/angular-nasa-apod.git
```

2. Instale as dependências:

```bash
npm install
```

3. Inicie o servidor de desenvolvimento:

```bash
npm start
```

---

## 🛠️ Scripts Disponíveis

- `npm start` - Inicia o servidor de desenvolvimento
- `npm run build` - Cria a build de produção
- `npm run serve` - Serve o aplicativo

---

O objetivo desse projeto foi aprender a usar o novo Express Angular Server Side Rendering (SSR) e reforçar meus conhecimentos em Angular.
